# [ubuntu 开机跳出“解锁密钥环”](https://www.cnblogs.com/yutian-blogs/p/13027241.html)

解决方法:

  如果开机不是自动登录，可以将密钥环密码设为和登录密码一致。这样登录后自动解密密钥环。

  如果是自动登录，可以将密钥环密码留空。

 

将密钥环设置为空的方法:

  \1. 用seahorse命令从命令行启动图形界面

  2.在左侧面板中，右击“默认密钥环”，并选择“修改密码”

  3.输入你的当前登录密码。

  4.密钥环新密码的密码框中留空

  5.在询问是否不加密存储密码对话框中点击“继续”
